## 重启计划

重启原因：原本只是本人开发前端，后端另外有人。现在各有各有的事忙起来不方便，改为本人全栈开发，由于涉猎较浅前后端全部采用nodejs相关语言开发。

代码结构划分

`router.js`文件：接口地址声明位置

`controller`文件夹：用于判断客户端数据和返回结果

`service`文件夹：用于处理业务操作，调用不同的查询语句

`model`文件夹：用于编写数据库的各种操作
